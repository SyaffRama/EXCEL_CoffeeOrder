## Overview
This project demonstrates **data analysis using Microsoft Excel**, covering data cleaning, lookup operations, and data visualization. The objective is to transform raw transactional data into structured insights using Excel formulas and dashboards.

## Dataset
- **Raw Data**: Coffee order transaction dataset  
- **Final Data**: Cleaned and enriched dataset prepared for analysis  
- **Content**: Customer details, product information, order volume, and sales metrics  
- Both raw and final files are included to clearly show the analytical process.

## Tools
- **Microsoft Excel**
  - INDEX-MATCH
  - Data cleaning techniques
  - Pivot tables
  - Charts & dashboards

## Steps
- Reviewed and explored raw data  
- Cleaned missing and inconsistent values  
- Applied `INDEX-MATCH` to combine and enrich datasets  
- Created pivot tables for aggregation and analysis  
- Built charts and dashboards to visualize trends and insights  

## Results
- Produced a clean, structured dataset  
- Built interactive dashboards for sales and order analysis  
- Identified trends in product performance and customer behavior  
- Delivered a clear, business-ready Excel analysis
